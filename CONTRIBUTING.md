# Contributing

## Development

1. Edit the code in `src/ytdlp_tui/`.
2. Run `./ytd-tui` locally in a terminal.
3. Verify the platform picker, search flow, queue, previews, and popups still render correctly.
4. Run a compile check before opening a pull request.

## Pull Requests

- Keep changes focused.
- Update the README when install steps, controls, or behavior change.
- If the Homebrew formula needs updates, update the tap repo as well.

## Releases

1. Commit changes.
2. Tag a new version such as `v1.0.1`.
3. Push the tag.
4. Update the Homebrew formula tarball URL and `sha256` when needed.
